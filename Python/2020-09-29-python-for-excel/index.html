通过观察课表，可以发现每一节课的内容大概是这样的：[课程名]◇任课教师[上课周数]◇[上课教室][第几节]。

其中[任课教师]和[上课周数]通常是成对出现，并且可能有多个，而[上课教师]和[第几节]通常成对出现。但需要注意的事，有的单元格内有些反常，例如星期日的下午 5-6 节课，这种需要进行针对性的处理。
