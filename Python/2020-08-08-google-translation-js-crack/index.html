然后分析一下请求 URL 所带的一些参数，参数有很多，不过只有 sl, tl, tk, q 是我们需要关注的，其中 sl 表示原文的语言类型，tl 表示译文的语言类型，tk 表示加密参数，需要破解，q 是我们输入的文本。
