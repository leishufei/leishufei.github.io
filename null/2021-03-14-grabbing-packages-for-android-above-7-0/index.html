Charles抓包安卓(7.0+)设备

发表于 2021-03-14 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

近期由于一些原因，需要抓取 APP 的数据。但安卓 7.0 以上版本的手机安装证书时只是安装到用户证书中，而非系统证书中，这就导致 Charles 抓包时无法抓到内容，所以有了这篇博客。

写在前面

本人手机型号为小米 Note3，安卓版本 10.0，其他安卓手机的操作方法应该和这类似。

刷机

  • 手机解锁

    下载解锁工具，然后跟着教程走。

    注意在 FastBoot 模式下用数据线连接手机时，可能会出现 Press any key to shutdown 字样，可参考这篇文章解决

  • 下载(开发版/测试版)线刷包

    到官网下载适合自己手机的 ROM 包(注意区分卡刷包和线刷包)，然后将下载好的压缩包解压以备后续使用。

  • 下载 MiFlash 工具

    到官网下载刷机工具

  • 刷机

    下载好上述工具后，然后跟着官方教程走，等待几分钟，即可完成刷机。

    刷机完后 MiFlash 中可能会出现 Mi Flash Error error:Not catch checkpoint ($fastboot -s .* lock),flash is not done，这不影响正常使用，可以忽略掉，参考这篇博客

手机 Root

刷到开发版/测试版后小米手机可以直接 Root，但从一些博客这个 Root 的权限似乎还不是最高的，如果有特别需要，可以借助 Magisk 进行 Root，具体方法 自行百度。

下载手机端证书

配置好 Charles 和手机 Wifi 代理，然后在手机端浏览器输入 chls.pro/ssl 下载一个后缀为 .pem 的证书文件，将其传输到 PC 端，为后续操作做准备。
